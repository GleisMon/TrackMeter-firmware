# TrackMeter Firmware

Firmware releases for the TrackMeter device (OTA updates).

The device checks `firmware/version.json` automatically and offers
the update in its web interface. No manual action is needed here.

| File | Purpose |
|------|---------|
| `firmware/version.json` | current release manifest (version, MD5, URL) |
| `firmware/TrackMeter.bin` | firmware binary |
